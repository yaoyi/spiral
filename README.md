spiral matrix
======

#### test

use rspec to run the test

```
  bundle install
  rspec
```

read the `spiral_spec.rb` for basic usage

#### example:

caculate the sum of the array below

```
  12 32 09 11 34
  08 54 76 23 07
  27 18 25 09 43
  11 23 78 63 19
  09 22 56 31 05
```

with follow clockwise spiral order

```
1  2  3  4  5
16 17 18 19 6
15 24 25 20 7
14 23 22 21 8
13 12 11 10 9 
```

run command to see the result
```
  ruby spiral_run.rb
```
